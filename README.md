### 背景

第97年第6月，永恒号遭遇高能射线暴。主数据库损毁，50名船员意识丢失。仅隔离节点7F（林汐终端）幸存。

事故调查报告结论：自然宇宙现象。

终端内的数据指向另一种可能。
